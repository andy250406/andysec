# '클로드 코드(Claude Code)' 소스 51만 줄 유출… "직원 실수"

## 📌 핵심 요약

앤트로픽의 AI 코딩 도구 '클로드 코드'의 원본 소스 코드 51만 줄이 NPM 배포 과정에서의 설정 실수로 유출되었습니다.

핵심 모델 가중치는 안전하나, 에이전트 설계 및 미공개 기능 등 내부 아키텍처가 노출되어 배포 프로세스 전반에 대한 재점검이 진행 중입니다.

## 🔍 주요 내용

* TypeScript 원본 소스 코드 약 51만 2,000줄과 1,900여 개의 파일이 NPM 소스 맵 노출을 통해 유출됨
* 빌드 설정 시 'cli.js.map' 파일을 포함하여 배포한 직원의 단순 휴먼 에러가 원인으로 파악됨
* 유출 본문에는 내부 에이전트 동작 원리, 미공개 모델 코드네임 및 백그라운드 자동화 관련 구동 로직이 포함됨
* 앤트로픽 측은 즉각적인 패키지 삭제와 저작권 침해 대응에 나섰으며 프로세스 재점검을 공식화함
## 단순한 설정 실수가 기술 경쟁력 유출로 이어지는 냉혹한 현실

이번 사고는 대형 AI 기업조차도 배포 프로세스의 작은 허점 하나로 핵심 기술 자산이 노출될 수 있음을 보여줍니다. 특히 소스 맵 유출은 컴파일 전의 원본 코드를 그대로 복구시켜주는 치명적인 통로가 됩니다. AI 개발 도구 시장의 선두주자로서 앤트로픽에게는 뼈아픈 실책이며, 이는 코드 보안이 단순히 코드 내부의 취약점 방어를 넘어 배포 인프라의 완결성까지 확장되어야 함을 시사합니다. 내부 로직의 투명한 공개가 경쟁사들의 벤치마킹 재료로 쓰이지 않도록 세밀한 통제가 필요합니다.
