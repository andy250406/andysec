# 의료 데이터를 위한 웹 취약점 자동 진단 시스템 개발

## 1. 개요
*   **수행 기간**: 2026.01 (SK루키즈 28기 미니 프로젝트 2)
*   **프로젝트 설명**: 민감한 의료 데이터를 처리하는 가상의 병원 시스템에 대해 주요 웹 취약점(OWASP Top 10)을 자동으로 진단하고 보고서를 출력해 주는 웹 자동 진단 시스템을 기획 및 구현했습니다.

## 2. 주요 기능
*   **OWASP Top 10 점검**: SQL Injection, XSS, 불완전한 인증 등에 대한 시뮬레이션 진단 로직 설계.
*   **리포트 자동 생성**: 취약점 점검 완료 시 등급(Critical, High, Medium, Low) 및 구체적인 대응 소스코드를 포함한 PDF/HTML 형식 리포트 추출.

## 3. 적용 기술
*   Python, Flask, JavaScript, SQL, Bootstrap (UI)
